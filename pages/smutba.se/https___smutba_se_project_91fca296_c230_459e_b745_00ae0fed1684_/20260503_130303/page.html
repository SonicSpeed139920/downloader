Fluid Painter 1.0

Starting from version 1.0, Fluid Painter NSFW is finally a proper addon!

Fluid Painter NSFW 1.0.1 Update

  • Added Slack parameters to Fluid String
  • Added the “Drop Starting Spline” parameter to define from which spline the drops are created
  • Added Resolution for Viewport and Render
  • Added Update Fluid Utility
  • Removed Thickness parameters and modified presets to reflect the change
  • Fixed a bug in the "Stick Fluid to Animated Object" tool occurring when the "Keep Fluid Properties" option was enabled in Blender 5.0
  • Fixed a bug in the Set Drawing Object function
  • Fixed a bug in the Geometry Node regarding curve tilt on fluid+ presets

Also check the What's New Document for more details about the new improvements and bug fixes!

Fluid Painter Compatibility

Fluid Painter NSFW is compatible with Fluid Painter on Superhive (Formerly known as Blender Market). After installing it along with the NSFW version, you'll get some additional features! For more details please check the document below, it also includes the compatibility table; since the two Fluid Painter versions must be paired with the right versions to work properly.

Fluid Painter + Fluid Painter NSFW

Description

Fluid Painter NSFW is a Blender add-on based on Geometry Nodes to paint fluids over any 3d surface using a curve.

Instructions

  1. In Blender, go to Edit > Preferences > Add-ons, select the zip file and enable the add-on.

  2. Open the Asset Browser and choose the Fluid Painter NSFW Library.

  3. Drag a preset in the scene, Initialize the fluid from the addon panel and Draw!

Be sure to check the images up there and the documentation for more info!
